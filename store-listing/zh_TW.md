無需離開 Chrome，即可將 OneSignal 管理儀表板瞬間翻譯成八種語言（日文、西班牙文、葡萄牙文、韓文、法文、土耳其文、簡體中文和繁體中文）。

本擴充功能通過一次點擊，將整個 OneSignal 介面（選單、按鈕、標籤、狀態徽章、彈出視窗、圖表和正文）本地化為您選擇的語言。專為在非英語市場透過 OneSignal 經營 Push 通知、Email、SMS 和應用內訊息活動的市場營銷人員、產品經理和 CRM 營運專業人員設計。

✦ 支援的語言
每種語言均完整覆蓋整個儀表板 — 包含超過 1,700 個儀表板字串以及用於日期格式、計數、百分比和相對時間標籤的 98 個動態模式：
• 日文（日本語）
• 西班牙文（Español，LATAM 通用）
• 葡萄牙文（Português Brasileiro / pt-BR）
• 韓文（한국어）
• 法文（Français）
• 土耳其文（Türkçe）
• 簡體中文
• 繁體中文

✦ 使用方法
1. 安裝本擴充功能並固定到 Chrome 工具列。
2. 開啟 dashboard.onesignal.com。
3. 點擊 OS Translate 圖示並選擇您的語言。
4. 儀表板會重新載入，所有可見文字將以您選擇的語言顯示。
5. 您的選擇在瀏覽器重新啟動後仍會保留，並在 Chrome 設定檔之間同步。

如需關閉翻譯，請在下拉選單中選擇「English (off)」。

✦ 隱私
本擴充功能僅在 dashboard.onesignal.com 上運行 — 不會影響其他任何網站。您的活動資料、受眾記錄、訊息內容、Segment 以及任何其他儀表板資料都不會離開您的裝置。

您也可以選擇共享一份匿名化的尚未翻譯儀表板字串列表，以便我們持續改進翻譯覆蓋率。僅會傳送英文原文和儀表板頁面名稱（例如 /apps/:id/journeys）。包含 ID 的 URL、客戶資料和個人資料絕不會被傳送。您可以隨時透過彈出視窗中的核取方塊關閉此選項。

✦ 回饋
右鍵點擊任何已翻譯的文字，選擇「為 ... 建議更好的翻譯」，即可開啟預先填好的回饋表單。或使用彈出視窗中的「回報翻譯問題」按鈕。提交內容會被定期審閱，並指導下一輪翻譯改進。
