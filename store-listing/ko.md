OneSignal 관리 대시보드를 7개 언어(일본어, 스페인어, 포르투갈어, 한국어, 프랑스어, 터키어, 간체 중국어)로 즉시 번역합니다. Chrome을 떠날 필요가 없습니다.

이 확장 프로그램은 OneSignal 인터페이스 전체(메뉴, 버튼, 라벨, 상태 배지, 모달, 차트, 본문)를 원하는 언어로 한 번의 클릭으로 현지화합니다. 비영어권 시장에서 OneSignal을 통해 푸시 알림, Email, SMS, In-App 메시징 캠페인을 운영하는 마케터, 프로덕트 매니저, CRM 운영 담당자를 위해 설계되었습니다.

✦ 지원 언어
각 언어는 대시보드 전체에 완전히 대응합니다 — 1,500개가 넘는 대시보드 문자열과 날짜 형식, 카운트, 백분율, 상대 시간 라벨을 위한 73개의 동적 패턴을 포함합니다:
• 일본어 (日本語)
• 스페인어 (Español, 라틴아메리카 중립)
• 포르투갈어 (Português Brasileiro / pt-BR)
• 한국어
• 프랑스어 (Français)
• 터키어 (Türkçe)
• 간체 중국어 (简体中文)

✦ 사용 방법
1. 확장 프로그램을 설치하고 Chrome 도구 모음에 고정하세요.
2. dashboard.onesignal.com을 엽니다.
3. OneSignal Translator 아이콘을 클릭하고 언어를 선택하세요.
4. 대시보드가 다시 로드되며 표시되는 모든 문자열이 선택한 언어로 표시됩니다.
5. 선택한 언어는 브라우저를 다시 시작해도 유지되며 Chrome 프로필 간에 동기화됩니다.

번역을 비활성화하려면 드롭다운에서 "English (off)"를 선택하세요.

✦ 개인정보 보호
이 확장 프로그램은 dashboard.onesignal.com에서만 동작합니다 — 다른 사이트에는 영향을 미치지 않습니다. 캠페인 데이터, 오디언스 기록, 메시지 콘텐츠, 세그먼트, 그 외 어떤 대시보드 데이터도 사용자의 컴퓨터를 떠나지 않습니다.

선택적으로, 번역 범위를 지속적으로 개선하기 위해 아직 번역되지 않은 대시보드 문자열의 익명화된 목록을 공유합니다. 전송되는 것은 영어 원문과 대시보드 페이지 이름(예: /apps/:id/journeys)뿐입니다. ID가 포함된 URL, 고객 데이터, 개인정보는 절대 전송되지 않습니다. 팝업의 체크박스를 통해 언제든지 이 옵션을 끌 수 있습니다.

✦ 피드백
번역된 텍스트를 마우스 오른쪽 버튼으로 클릭하고 "더 나은 번역 제안..."을 선택하면 미리 입력된 피드백 양식이 열립니다. 또는 팝업의 "번역 문제 신고" 버튼을 사용하세요. 제출된 내용은 정기적으로 검토되어 다음 번역 개선에 반영됩니다.

이 확장 프로그램은 커뮤니티 주도로 개발되었으며 무료로 사용할 수 있습니다.
