无需离开 Chrome，即可将 OneSignal 管理仪表板瞬时翻译成八种语言（日语、西班牙语、葡萄牙语、韩语、法语、土耳其语、简体中文和繁体中文）。

本扩展通过一次点击，将整个 OneSignal 界面（菜单、按钮、标签、状态徽章、模态窗口、图表和正文）本地化为您选择的语言。专为在非英语市场通过 OneSignal 运营 Push 通知、Email、SMS 和应用内消息活动的市场营销人员、产品经理和 CRM 运营专业人员设计。

✦ 支持的语言
每种语言均完整覆盖整个仪表板 — 包含超过 1,700 个仪表板字符串以及用于日期格式、计数、百分比和相对时间标签的 98 个动态模式：
• 日语（日本語）
• 西班牙语（Español，拉美中立）
• 葡萄牙语（Português Brasileiro / pt-BR）
• 韩语（한국어）
• 法语（Français）
• 土耳其语（Türkçe）
• 简体中文
• 繁体中文，HK（繁體中文）

✦ 使用方法
1. 安装本扩展并将其固定到 Chrome 工具栏。
2. 打开 dashboard.onesignal.com。
3. 点击 OS Translate 图标并选择您的语言。
4. 仪表板重新加载，所有可见文本将以您选择的语言显示。
5. 您的选择在浏览器重启后仍会保留，并在 Chrome 配置文件之间同步。

如需关闭翻译，请在下拉菜单中选择 "English (off)"。

✦ 隐私
本扩展仅在 dashboard.onesignal.com 上运行 — 不会影响其他任何网站。您的活动数据、受众记录、消息内容、Segment 以及任何其他仪表板数据都不会离开您的设备。

您也可以选择共享一份匿名化的尚未翻译的仪表板字符串列表，以便我们持续改进翻译覆盖率。仅会发送英文原文和仪表板页面名称（例如 /apps/:id/journeys）。包含 ID 的 URL、客户数据和个人信息绝不会被传输。您可以随时通过弹出窗口中的复选框关闭此选项。

✦ 反馈
右键单击任何已翻译的文本，选择 "为 ... 推荐更好的翻译"，即可打开预先填好的反馈表单。或使用弹出窗口中的 "报告翻译问题" 按钮。提交内容会被定期审阅，并指导下一轮翻译改进。
